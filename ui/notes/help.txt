----basic info----
help: to show help
exit: to exit
modules: to show modules
use <module>: to use module
back: to return back or exit (in case no module choosed)

----modules info----
set "parameter": to set needed parameter
show stuff: to show a table of module
run: to run module
ecmd: to enable cmdline
dcmd: to disable cmdline

----extra info----
(R): required
(O): optional
(Re): recommanded
(In): instable
